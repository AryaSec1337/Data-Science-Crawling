# Instalasi 🚀
Sudah pasti require Python , install Python terlebih dahulu: [Install Python](https://www.python.org/downloads/)

```

pip install matplotlib
pip install jupyterlab
jupyter notebook
//jika anda root
jupyter notebook --allow-root
```
